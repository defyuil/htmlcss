background - 배경

  1. background-color:색상값; 배경색상
  2. background-image:url("이미지경로"); 배경이미지

background-image 2개 지정

background-image:url("이미지1경로"), url("이미지2경로");

background-image의 속성들

  • background-repeat:repeat/repeat-x/repeat-y/no-repeat; 반복 지정
    1. repeat (기본) 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 반복 안함
  • background-position: ; 위치 지정
    1. left/center/right top/center/bottom 의 조합
    2. Xn Yn - 가로 세로 순서로 직접 지정
  • background-size: ; 크기 지정
    1. cover - 이미지 하나로 영역을 다덮기
      (이미지 잘려나갈 수도 있음)
    2. contain - 영역의 가로나 세로에 맞춰 이미지로 덮기
      (이미지 잘려나가지 않음)
    3. Xn Yn - 가로 세로 순서로 직접 지정
  • background-attachment: ; 스크롤 여부 지정
    1. scroll - (기본)스크롤
    2. fixed - 고정
    3. local - 영역에서...

background-attachment: ; 스크롤 여부

background-attachment:scroll; - 특정영역의 스크롤바에는 고정, 브라우저 스크롤바에는 스크롤
background-attachment:fixed; - 특정영역의 스크롤바와 브라우저에 모두 고정
background-attachment:local; - 특정영역의 스크롤바와 브라우저에 모두 스크롤
