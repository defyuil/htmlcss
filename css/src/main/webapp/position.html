position 위치 지정 방식

  • position:static; (기본)소스 순서대로 정렬
  • position:relative; 현 위치에서 ~거리 지정
  • position:absolute; 상위요소를 기준으로 ~거리 지정
  • position:fixed; 브라우저를 기준으로 ~ 고정
  • position:sticky; relative와 fixed 믹스

거리값 지정

  • left/right:n; 수평
  • top/bottom:n; 수직

5. position:sticky; relative와 fixed믹스

브라우저의 특정위치에 도착하면 고정. sticky를 적용할 때 다른 상위요소가 없어야 함
* 색상 적용시 투명도 조절 = rgba(n,n,n,n) a(alpha)는 불투명도 0~1사이의 값.

1. position:static; (기본)소스 순서대로 정렬

position:static; left: 500px; 고정

2. position:relative; 현 위치에서 ~거리 지정

position:relative; left:500px;
position:relative; right:500px;
position:relative; top:500px;
position:relative; bottom:500px;
position:relative; left: 900px; bottom: 300px;
position:relative; right: 200px; bottom: 200px;

3. position:absolute; 상위요소를 기준으로 ~거리 지정 (기준이 될 상위요소에 position: relative 설정)

position:absolute; left: 500px;
position:absolute; right:500px;
position:absolute; top:300px;
position:absolute; bottom:30px;
